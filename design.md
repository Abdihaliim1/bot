# Design Style Guide - Telegram Video Downloader Bot Landing Page

## Design Philosophy

### Color Palette
- **Primary**: Deep Teal (#1a5f5f) - Professional, tech-focused
- **Secondary**: Warm Coral (#ff6b6b) - Energy and engagement
- **Accent**: Soft Mint (#4ecdc4) - Fresh, modern touch
- **Neutral**: Charcoal Gray (#2c3e50) - Text and contrast
- **Background**: Off-white (#fafafa) - Clean, minimal base

### Typography
- **Display Font**: "Canela" - Bold serif for headlines and hero text
- **Body Font**: "Suisse Int'l" - Clean sans-serif for readability
- **Code Font**: "JetBrains Mono" - For technical elements

### Visual Language
- Modern tech aesthetic with editorial sophistication
- Clean geometric shapes and subtle gradients
- High contrast for accessibility
- Minimal but impactful visual elements
- Focus on functionality and user experience

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **p5.js**: Creative coding for hero background effects
- **ECharts.js**: Data visualization for bot statistics
- **Splide.js**: Smooth carousel for feature showcases
- **Shader-park**: Subtle background visual effects

### Animation Effects
- **Hero Section**: Floating particles with p5.js creating dynamic background
- **Text Effects**: Color cycling emphasis on key phrases using Anime.js
- **Card Animations**: Subtle hover transforms and shadow effects
- **Loading States**: Smooth progress indicators for bot interactions
- **Scroll Animations**: Gentle reveal animations as content enters viewport

### Header Effect
- Clean navigation with subtle backdrop blur
- Smooth color transitions on scroll
- Minimal shadow for depth without heaviness

### Interactive Elements
- Smooth button hover states with color morphing
- Card lift effects on hover
- Progressive disclosure for feature details
- Responsive feedback for all interactive elements

## Layout Principles
- Grid-based design for consistency
- Generous whitespace for breathing room
- Mobile-first responsive approach
- Content hierarchy through typography and spacing
- Accessibility-first color contrast ratios
